Pertanyaan Sering Diajukan

Apakah SND App gratis?
Yap! Saat ini semua tools di SND App gratis digunakan untuk komunitas Engineer dan Drafter FTTH Indonesia.
Data saya aman gak?
Sangat aman. SND App memproses data sepenuhnya di browser Anda (Client-Side). File KMZ/Excel Anda tidak pernah diunggah ke server mana pun.
Bagaimana cara request fitur custom?
Silakan gunakan form di atas dan pilih kategori "Request Fitur Baru". Jelaskan kebutuhan Anda sedetail mungkin (screenshot atau contoh file sangat membantu).
Support format apa saja?
Fokus utama kami adalah format .KML dan .KMZ (Google Earth). Output yang dihasilkan bervariasi mulai dari CSV, Excel, hingga GeoJSON.
